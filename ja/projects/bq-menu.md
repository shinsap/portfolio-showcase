# BQ Menu

## 概要

BQ Menuは、営業先での説明やデモに使うクローズドなWebアプリです。完全公開のプロダクトではなく、商談や説明のためのデモ環境として運用しています。

## 担当範囲

- デモフロー設計
- UI実装
- Vercel向けデプロイ設定
- security headers / routing設定
- CI / gitleaks workflowの整備

## 技術スタック

- Frontend web app
- Vercel
- GitHub Actions

## 伝えられること

- 営業デモを意識したプロダクト設計
- 軽量なデプロイ構成
- security headersの設定
- CI checkの整備
