# 電池あった？

## 概要

電池あった？は、個人・家庭向けに電池を記録しておくPWAです。家の中にある機器ごとに、どの電池が使われているかを記録し、電池交換の時にすぐ確認できるようにすることを目的にしています。

補聴器でんち帳とは別のプロダクトです。電池あった？は家庭内の電池管理に特化しており、介護施設向けの補聴器電池記録業務とは用途を分けています。

## 担当範囲

- プロダクト企画
- 家庭内電池記録フローの設計
- PWA前提の実装検討
- local-firstデータ管理方針の整理
- 電池交換時の確認UX設計

## 技術スタック

- PWA
- React
- TypeScript
- Vite
- React Router
- Dexie
- Firebase
- vite-plugin-pwa

## 利用シーン

家庭内には、リモコン、小型機器、日用品など、さまざまな種類の電池を使うものがあります。電池あった？は、機器ごとの使用電池を記録しておき、交換時に「どの電池だったか」を確認できるようにするアプリです。

## 伝えられること

- 個人利用向けPWA設計
- 軽量な記録・確認ワークフロー
- local-firstのプロダクト設計
- 家庭内の繰り返し作業に向けた実用的なUX設計
