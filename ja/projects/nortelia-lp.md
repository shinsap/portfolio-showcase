# Nortelia LP

## 概要

Nortelia LPは、サービスやブランドの価値を伝えるための公開向けランディングページです。

## 担当範囲

- ページ構成の設計
- ビジュアル方向性の整理
- フロントエンド実装
- コピー・コンテンツ整理
- デプロイ前提のbuild check整備

## 技術スタック

- Modern frontend stack
- GitHub Actions build check
- Gitleaks workflow

## 伝えられること

- LPの設計と実装
- 公開向けページの見せ方
- コンテンツを中心にした構成
- 基本的なCIとrepo hygiene

## ポートフォリオ用途

privateなプロダクト内部を見せずに、実際のユーザー向け成果物を見せられるため、応募時のポートフォリオURLとして使いやすいプロジェクトです。
