# Nortelia LP

## 概要

Nortelia LPは、サービスやブランドの価値を伝えるための公開向けランディングページです。

このLP内には、補聴器でんち帳のLPも内包されています。補聴器でんち帳LPは補聴器の電池検索ができるページになっており、電池検索を入口として補聴器でんち帳PWAへつなぐ役割を持っています。

## 担当範囲

- ページ構成の設計
- ビジュアル方向性の整理
- フロントエンド実装
- コピー・コンテンツ整理
- 補聴器でんち帳のプロダクト導線設計
- 補聴器電池検索LPの導線設計
- デプロイ前提のbuild check整備

## 技術スタック

- Modern frontend stack
- GitHub Actions build check
- Gitleaks workflow

## 伝えられること

- LPの設計と実装
- 公開向けページの見せ方
- コンテンツを中心にした構成
- LP内にサービス導線を組み込む設計
- 検索からPWAへつなぐ導線設計
- 基本的なCIとrepo hygiene

## ポートフォリオ用途

privateなプロダクト内部を見せずに、実際のユーザー向け成果物を見せられるため、応募時のポートフォリオURLとして使いやすいプロジェクトです。
