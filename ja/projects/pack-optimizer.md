# Pack Optimizer

## 概要

Pack Optimizerは、不定貫商材の固定個数パックを目標重量範囲内で作るための公開用最適化ツールです。現在はVercelで運用しています。

GitHub: https://github.com/shinsap/pack-optimizer

## 公開URL

- App: https://pack-optimizer.vercel.app/

## 担当範囲

- ツール企画
- Next.jsフロントエンド実装
- 不定貫商材向けの固定個数パック最適化ロジック
- Vercelデプロイ設定
- リポジトリ保守

## 技術スタック

- Next.js
- React
- TypeScript
- Tailwind CSS
- Vercel

## 保守メモ

現在のVercelデプロイは `pack-optimizer` リポジトリに接続されています。古い `kombu` リポジトリは現在の公開プロジェクトではないため、ポートフォリオから外しています。

## 伝えられること

- 小規模ユーティリティ開発
- public repository運用
- 不定貫商材向けの固定個数パック最適化
- Next.jsアプリのVercel運用
