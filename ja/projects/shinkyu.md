# Shinkyu

## 概要

Shinkyuは、鍼灸治療院向けの営業支援ツールです。

## 担当範囲

- 要件整理
- UI / workflow設計
- Cloudflare向けデプロイ設定
- アプリケーション実装支援

## 技術スタック

- Next.js
- React
- TypeScript
- Supabase
- Anthropic SDK
- Tailwind CSS
- Cloudflare Pages
- @cloudflare/next-on-pages
- Wrangler

## 伝えられること

- 業務支援ツールの設計
- Cloudflare deployment configuration
- 速いプロトタイピングと改善
