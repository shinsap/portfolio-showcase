# BQ Menu

## Overview

BQ Menu is a labor-saving menu display system for hotel banquet venues. Guests or staff scan a QR code to view the menu for a banquet or event, reducing the need for printed materials and manual explanation.

The project is currently operated as a closed sales-demo environment rather than a fully public product.

## Role

- Demo flow planning
- QR-based menu flow design
- UI implementation
- Vercel-oriented deployment setup
- Security header and routing configuration
- CI and gitleaks workflow maintenance

## Stack

- Frontend web app
- Vercel
- GitHub Actions
- QR menu display flow

## Notes

The project is operated in a closed/demo context for business discussions with potential customers. Source code and live access are not broadly public, but the project is useful as an example of a focused operational tool for hospitality workflows.

## What This Demonstrates

- Hotel banquet workflow understanding
- QR-based guest/staff menu access
- Labor-saving operational tool design
- Lightweight deployment setup
- Practical security headers
