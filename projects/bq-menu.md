# BQ Menu

## Overview

BQ Menu is a closed sales-demo web application. It is used as a demo environment for business conversations rather than a fully public product.

## Role

- Demo flow planning
- UI implementation
- Vercel-oriented deployment setup
- Security header and routing configuration
- CI and gitleaks workflow maintenance

## Stack

- Frontend web app
- Vercel
- GitHub Actions

## Notes

The project is operated in a closed/demo context. Source code and live access are not broadly public, but the project is useful as an example of demo-oriented application development.

## What This Demonstrates

- Business-demo product thinking
- Lightweight deployment setup
- Practical security headers
- Maintained CI checks
