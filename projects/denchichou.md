# Denchichou

## Overview

Denchichou is a local-first PWA for care workers to record hearing-aid battery replacement activity.

## Role

- Product planning
- Workflow design
- Local-first application direction
- PWA-oriented implementation planning
- Firebase-related configuration review

## Stack

- Local-first PWA architecture
- Firebase-related configuration
- Modern frontend development

## What This Demonstrates

- Domain-specific operational tool design
- Offline/local-first thinking
- Practical workflow modeling for repeated daily work
