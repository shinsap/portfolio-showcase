# Rescue Cat Card LP

## Overview

Rescue Cat Card LP is the public-facing landing page for Rescue Cat Card. It explains the service concept, use case, and path to registration for people who want emergency pet-care information to be reachable when something happens to them.

The LP is separate from the application implementation. Its role is to communicate the value of the service clearly, reduce hesitation, and route users toward the Rescue Cat Card product.

## Public URL

- LP: https://rescuecat.jp/

## Role

- Landing page structure planning
- Service explanation and user-flow organization
- Copy and section design
- Registration/conversion path planning
- Public-facing content review

## Stack

- Modern frontend stack
- Landing page implementation
- Public content structure
- Deployment-oriented build checks

## Relationship to Rescue Cat Card v4

Rescue Cat Card v4 is the launched service application with authentication, user records, QR/card-related flows, Supabase, and Cloudflare-based operation.

Rescue Cat Card LP is the acquisition and explanation layer. It presents the service to new users and creates a clean path toward the application.

## What This Demonstrates

- Service landing page design
- Public product communication
- Conversion-oriented page structure
- Separation between product application and marketing surface
